# Project Title:

Daily Check-in Tracker (Python CLI)

# What this does:

A simple command-line app that asks daily habit questions and gives motivational feedback.
Built to practice Python basics like input, conditionals, and strings.

# How to Run:
Step 1: Install Python

Download from python.org
Make sure python --version works in terminal

Step 2: Download the project

Click “Code” → Download ZIP
Or clone the repo

Step 3: Run the file
In terminal:
      cd Daily_Check_In_Tracker
python main.py

The code will run after this..
